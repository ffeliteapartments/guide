🔐 Sicurezza
Cambia PIN Host

Il PIN predefinito è 1234. Cambialo subito per proteggere le impostazioni.

🔑 Cambia Username / Password

Le credenziali di default sono admin / admin. Cambiale per proteggere il tuo account.

🔑 Parola Segreta di Recovery

La parola segreta permette di resettare il PIN in caso di smarrimento. La parola segreta è stata configurata dall'amministratore al setup iniziale.

🚀 Pubblica Online

⚠️ Usa un fine-grained token con accesso solo a questo repository. Il token è cifrato e salvato in modo sicuro nel browser.

💡 Cliccando "Salva Token per Host", l'host potrà pubblicare direttamente con il pulsante "🚀 Pubblica Ora" senza conoscere il token.

🔗 Dominio Personalizzato

Configura un dominio personalizzato (es. guida.miobnb.it) per la tua guida su GitHub Pages.

📖 Come configurare il dominio
  1. Acquista un dominio (es. su Aruba, Namecheap, Porkbun…)
  2. Vai nelle impostazioni DNS del tuo provider
  3. Aggiungi un record CNAME che punta a <username>.github.io (clicca il pulsante sopra per copiarlo)
  4. Vai su GitHub repo → Settings → Pages → Custom domain
  5. Inserisci il dominio e salva
  6. Attendi la propagazione DNS (fino a 24h)
  7. Abilita "Enforce HTTPS"
